Marginalia - FAQ

Purpose: A distraction-free drafting environment designed for writers who draft from existing outlines and chapter cards.

Features

Import Cards

Import a chapter card file. Cards remain separate from your manuscript and act as reference material while drafting.

Import / Export MD

Import an existing markdown manuscript or export your current draft as a .md file for Word, Atticus, or other writing software.

Import / Export Project

Project files save and restore your manuscript, cards, theme, and workspace settings.

Focus Mode

Hide the outline pane and write with fewer distractions.

Autosave & Restore

Your work is automatically saved locally and restored when reopening the application.

Themes

Switch between Dark Draft, Paper Draft, Manuscript, and E-Ink modes.

Formatting & Markdown

Marginalia uses simple Markdown formatting because it is lightweight, reliable, and supported by most modern writing tools.

  • Chapter inserts a Markdown chapter heading.
  • Page Break inserts a page break marker for export.
  • B wraps selected text in **bold**.
  • I wraps selected text in *italic*.
  • L, C, and R wrap selected text with alignment tags.

Do not worry about seeing Markdown symbols while drafting.
